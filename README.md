# MangroveGuard
**"Aplikasi Konservasi Mangrove Berbasis Komunitas (Citizen Science)"**

## Overview Produk
> MangroveGuard adalah aplikasi mobile berbasis komunitas (Citizen Science) yang dirancang untuk memantau dan mendokumentasikan kondisi ekosistem mangrove secara real-time. Aplikasi ini memberdayakan konservasionis, peneliti, dan masyarakat umum untuk berpartisipasi aktif dalam perlindungan hutan mangrove dengan merekam data lapangan seperti observasi plot, polusi, dan gangguan.

> **Fungsi Utama:**
> - Melakukan Observasi Plot: Mencatat data detail tentang jenis, kesehatan, dan kerapatan mangrove.
> - Mendokumentasikan Ancaman: Merekam kejadian polusi (misal tumpahan minyak, sampah plastik) dan gangguan (misal penebangan ilegal).
> - Mengidentifikasi Spesies: Mengakses panduan mangrove untuk identifikasi cepat.
> - Memvisualisasikan Data: Menampilkan semua rekaman data dan plot observasi pada peta proyek interaktif.

> **Tujuan:**
> Menyediakan data terverifikasi untuk mendukung mitigasi, pengambilan keputusan konservasi, dan meningkatkan kesadaran publik terhadap pentingnya ekosistem mangrove.

### 1. Sistem Pencatatan Data Lapangan
> Fitur inti untuk merekam berbagai jenis data lapangan:
> - **Pencatatan Observasi:** Mencatat kondisi plot, termasuk jenis mangrove (misal *Rhizophora mucronata*), DBH, tinggi, kerapatan, kondisi kesehatan (Sehat, Stres, Mati), dan substrat (Lumpur, Pasir, Campuran).
> - **Pencatatan Polusi:** Merekam jenis polusi (misal sampah plastik, tumpahan minyak), tingkat keparahan (Ringan, Sedang, Berat), dan luas/sebaran area terdampak.
> - **Pencatatan Gangguan:** Dokumentasi jenis gangguan (misal penebangan ilegal), skala kerusakan (Kecil, Sedang, Luas), dan perkiraan area terdampak (mÂ²).
> - **Pencatatan Lain-Lain (Other Sightings):** Merekam penampakan hewan atau spesies langka di area mangrove.
> - **Ringkasan Data:** Halaman utama menampilkan statistik total Observasi, Polusi, Gangguan, dan Other Sightings yang sudah direkam.

### 2. Panduan Mangrove (Learn)
> Fitur identifikasi spesies, menyajikan informasi seperti:
> - Nama ilmiah (misal *Acrostichum aureum*)
> - Status konservasi (misal LC â€“ Least Concern)
> - Distribusi alami
> - Nama umum (vernacular names)
> - Tipe tanaman

### 3. Peta Proyek Interaktif
> Memvisualisasikan lokasi semua plot observasi dan titik data lain (polusi, gangguan) di peta interaktif.  
> Mendukung tampilan: **STANDARD**, **SATELLITE**, dan **TERRAIN**.

### 4. Manajemen Data Pengguna (My Records)
> Menampilkan daftar lengkap data yang telah direkam oleh pengguna, lengkap dengan detail waktu, koordinat, dan deskripsi.  
> Terdapat opsi: **EDIT**, **HAPUS**, dan melihat lokasi rekaman di Google Maps.
